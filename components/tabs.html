    # Tabs Attributes

    参数 说明 类型 可选值 默认值
    value / v-mode 绑定值，选中选项卡的 name string - -

    # Tabs Events

    事件名称 说明 回调参数
    tab-click tab 被选中时触发 被选中的标签 name

    # TabsNav Attributes

    参数 说明 类型 可选值 默认值
    name 与选项卡绑定值 value 对应的标识符，表示选项卡别名 string - -
    disabled 是否禁用 boolean - false

    # TabsPane Attributes

    参数 说明 类型 可选值 默认值
    name 与选项卡绑定值 value 对应的标识符，表示选项卡别名 string - -
    label 是否禁用 boolean - false
